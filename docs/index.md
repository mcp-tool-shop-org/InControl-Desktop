# InControl Desktop

Privacy-first, GPU-accelerated local AI chat assistant for Windows.

## Key Features

- **Private by Default** — Conversations never leave your machine
- **RTX-Optimized** — CUDA acceleration for fast local inference
- **Native Windows** — WinUI 3 with Fluent Design
- **Multiple Backends** — Ollama, llama.cpp, or custom inference engines
- **Markdown Rendering** — Syntax highlighting and rich output

## NuGet Packages

```bash
dotnet add package InControl.Core
dotnet add package InControl.Inference
```
